# 📐 Rectangle Area Calculator

This C# console application calculates the area, perimeter, and diagonal length of a rectangle based on user input.

---

## ✨ Features

- Takes width and height input from the user
- Calculates and displays:
  - Area of the rectangle
  - Perimeter of the rectangle
  - Length of the rectangle's diagonal

---

## 🛠️ Concepts Covered

- Creating classes with methods in C#
- Using `Math` library functions (`Math.Sqrt`)
- Console input and output
